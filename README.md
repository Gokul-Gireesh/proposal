# proposal | Gokul Gireesh

A collection of technical proposals, language specifications, and architectural drafts aimed at pushing the boundaries of modern computing.

## 🚀 Current Proposals

### 1. [hinted.js](./hinted.js/)
**Category:** Programming Languages / ECMAScript  
**Status:** Draft (Stage 0)  
**Vision:** Bringing near-native memory management to JavaScript through blind engine optimization hints (`int 4`, `bit`, `str 3`). 

### 2. [private-branches](./private-branches/)
**Category:** Version Control / GitHub Infrastructure  
**Status:** Feature Request  
**Vision:** Selective Branch Publishing & Public Fork Control. Allow repository owners to designate specific branches as "Public" for forks while keeping internal development branches private within the same repo.
